# Competitor gap audit framework

Use this framework every month to find ranking gaps against battery, fitment,
and roadside competitors around Alberton.

## Competitors to monitor

Track local competitors that appear for:

- battery replacement Alberton
- car battery Alberton
- mobile battery replacement Alberton
- battery fitment Meyersdal
- truck battery Alrode
- Willard battery Alberton
- 652 battery Alberton
- BMW battery coding Meyersdal

## SERP collection checklist

For each keyword, record:

| Field | What to capture |
| --- | --- |
| Keyword | Exact query searched |
| Competitor URL | Ranking page, not just domain |
| Page type | Home, service, local, product, blog, GBP |
| Intent covered | Emergency, price, testing, fitment, brand, vehicle |
| Proof signals | Photos, reviews, local landmarks, before/after work |
| Structured data | Product, LocalBusiness, FAQ, Article, Service |
| Weakness | Thin copy, no suburb proof, no CTA, outdated info |
| ABM response | Existing page to improve or new page target |

## Page gap rules

Create or improve a page only when at least one of these is true:

1. Competitors rank with a page that maps to a clear service, suburb, vehicle,
   brand, or battery-code intent.
2. The query has commercial value: callout, quote, testing, replacement,
   fitment, coding, fleet, AGM/EFB, truck, or brand/code terms.
3. Alberton Battery Mart can add unique local proof: photos, suburb routes,
   vehicle examples, diagnostic workflow, or customer reviews.

Avoid creating pages that only swap a suburb name with no unique value.

## Priority response matrix

| Competitor advantage | ABM response |
| --- | --- |
| More Google reviews | Push `/reviews` after every job and reply to every review. |
| Better GBP photos | Upload weekly fitment, stock, shopfront, fleet, and technician photos. |
| Ranking for suburb service | Add or strengthen `/services/[service]/[area]`. |
| Ranking for vehicle query | Strengthen `/vehicles/[make]/[model]` and add proof page if real job evidence exists. |
| Ranking for battery code | Strengthen `/products/size/[code]` with service links and real stock info. |
| Ranking for brand query | Strengthen `/products/brand/[brandName]` and link to service pages. |

## Monthly execution loop

1. Pull Search Console queries and pages.
2. Search top 20 money keywords manually in an incognito/localized session.
3. Add findings to the matrix above.
4. Pick the top 5 gaps by commercial intent.
5. Update pages, internal links, and GBP posts.
6. Add real proof photos/reviews to the matching proof pages.
7. Validate indexing and rich-result reports after deployment.
